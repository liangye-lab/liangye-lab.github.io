从高温往下，cluster algorithm是怎样失效的

实验背景和目的

本实验讨论的是$\kappa=0.07$高于PF相变温度的区域。在这个区域cluster算法失效。但是显然在高温区，cluster算法应该是可以用的。所以现在我们探讨的问题是：

Cluster算法从什么温度开始失效？从介观结构上，失效时发生了什么？这些变化如何影响cluster算法？

实验设计

实验1 求cluster算法的$\tau-T$图

我们可以计算E和M的$\tau$。预期随着温度降低，$\tau$开始升高。显然，计算$\tau$的时候要求采样达到平衡，而$\tau$和$\tau_{eq}$应该存在关联，所以到某个温度的时候$\tau$高到无法平衡，我们需要知道这个点在哪里。

我预期dynamic critical exponent应该是没法严格算的，因为在远离critical point的时候算法就失效了。当然或许可以外推。

实验2 随着温度降低、$\tau$增长，系统的介观结构的变化

观察movies，可以借助计算cluster size distribution等来考察系统的变化。我现在不知道什么样的现象能够将算法和模型性质联系起来，只能随便猜测。
