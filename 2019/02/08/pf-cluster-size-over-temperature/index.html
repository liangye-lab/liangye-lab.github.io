PF线Wolff算法产生的cluster尺寸观察

实验背景和目的

之前的FSC实验需要提高$U_4$精度，也就是说要么提高采样量，要么提高采样效率。然而，在PF线上Wolff算法竟然也出现了采样效率低的现象（确定Ising点效率还比较高）。我们需要了解在这个区域本来预期效率不差的Wolff算法究竟发生了什么。

我一直认为，ANNNI中cluster算法可用，是因为其阻挫比较弱，一个粒子的8条键中只占2条；而WW模型的阻挫特别强，占到18/24（如果将$\gamma=2$视作数两次，那么就是30/36）。我试图观察是不是因为阻挫的显著增加而导致出现了Wolff算大明显的变化。
