March 2, 2019 · 月度总结

2019年二月总结

二月的项目还是WW模型。总体来说，二月份主要做了两件事：说明Wolff类的cluster算法对WW模型不是很好用，以及写Prelim的文档。

工作方向

现在有着比较充分的理由认为Wolff类型的cluster算法不是很好用，只能勉强算超低$\kappa$的有限尺度标度。显然这样是不够的。所以下面的方向是搜索替代算法。

写document花了两周，几乎是全天工作。这次的教训是document最好在工作时间外写，并且给更多的时间。对于后面的slides，应该马上开始搭建并成形，但是不能占用白天的工作时间。

工作状态

二月初的状态不是特别好。当时一直没有理解导师的意图，所以工作得很犹豫。在给出PF的cluster尺寸的图之后就比较顺畅了，当然也就开始写document了。

对于写document，导师看上去也比较理解。当然导师给了很多建议，对此我心存感激。写document的第一周，写作效率很低。直到当周的周四熬了一夜，把很多内容重写了，周五中午才交初稿。周五导师发回意见，认为需要大调结构，但是周末却没有去动。直到第二周周一周二才调整完。之后导师说不需要他来改了，意味着大框架完成了，总体上应该是没有问题的。当然导师也提了一些意见，特别是针对motivation。这一点如果在后面评审的时候有出现疑问，我可以再补充。周四调了调格式，周五交了。当然中间其实还做了一个小实验，补了一个Wolff算法的小洞。总体来说第二周效率更高、工作强度更大一点。

总结

总之，现在研究方向也明确了，知道了自己该看什么做什么，而且slides也需要做了。每天的计划很清楚了，希望三月份充实、顺利。
